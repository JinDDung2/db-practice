# 내친

*** 

* Facebook, Instagram, Twitter와 같은 SNS 서비스 ‘내친’을 개발하는 프로젝트
* 프론트엔드 부분은 생략하고 벡엔드에 초점을 맞춰 백엔드 개발에 주력
* 단순 기능 구현 뿐만 아니라, 코드의 재사용성, 트래픽 고려 및 유지보수성을 고려하여 구현하는 것을 목표로 개발

***
